## Deployment

This site is deployed on [Netlify](https://www.netlify.com/) with automatic deployments.

### Deployment Configuration

The `netlify.toml` file includes:
- **Publish directory:** `public`
- **Security headers:** XSS protection, frame options, content type options
- **Performance caching:** 1-year cache for CSS, JS, and images
- **SPA routing:** All routes redirect to main HTML file

### Deploy Your Own

1. Fork this repository
2. Log in to [Netlify](https://app.netlify.com/)
3. Click "Add new site" ‚Üí "Import an existing project"
4. Connect your GitHub account and select your forked repository
5. Netlify will auto-detect settings from `netlify.toml`
6. Click "Deploy site"

Your site will be live at a custom Netlify URL!
